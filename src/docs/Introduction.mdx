# Applications Showcase

A living component library showcasing UI components from 9 full-stack applications, each built with different technology stacks and design approaches.

## Applications

### DevOps & Infrastructure

| App | Description | Stack |
|-----|-------------|-------|
| **PodRick** | DevOps portfolio manager — centralized dashboard for deploying and monitoring applications via ArgoCD, Helm, and GitHub Actions | React 19, Pure CSS, CSS Variables |
| **TenantFlow** | Kubernetes multi-tenant platform — manage tenants, deployments, quotas, and topology across a K8s cluster | React 19, Pure CSS, Chart.js |
| **Pop!\_Portfolio** | Portfolio orchestration dashboard — unified control plane for all portfolio microservices with real-time monitoring | React 19, MUI v7, Emotion, Socket.IO |

### Full-Stack Applications

| App | Description | Stack |
|-----|-------------|-------|
| **FireBook** | AI-powered bookmark manager — auto-tagging, semantic search, AI descriptions, and similar bookmark discovery powered by Firebase Cloud Functions | React 19, Tailwind CSS, Firebase |
| **Bookmarked** | Bookmark management with AI features — semantic search, virtual scrolling, batch tag generation, and smart filtering | React 18, Emotion, Express |
| **educationELLy** | Student management system — CRUD operations, pagination, AI chat assistant, and responsive dashboard | React 18, styled-components, Semantic UI, Redux |
| **educationELLy-GraphQL** | GraphQL-powered education platform — Apollo Client with real-time queries, mutations, and student management | React 18, styled-components, Semantic UI, Apollo Client |
| **Code Talk** | Real-time code collaboration — collaborative text editor with live messaging, room management, and WebSocket subscriptions | React 18, styled-components, Apollo Client, WebSockets |

### AI & Machine Learning

| App | Description | Stack |
|-----|-------------|-------|
| **IntervalAI** | Neural-enhanced spaced repetition — TensorFlow.js ML models, D3 neural network visualization, and adaptive learning algorithms | React 18, CSS Modules, Redux, D3, TensorFlow.js |

## Technology Coverage

This showcase demonstrates proficiency across:

- **6 styling systems**: Tailwind CSS, Emotion, styled-components, MUI, CSS Modules, Pure CSS
- **4 state management approaches**: Redux Toolkit, Apollo Client, Firebase Context, React local state
- **Real-time features**: WebSocket subscriptions, live collaboration, auto-refresh polling
- **AI/ML integration**: TensorFlow.js, Firebase AI functions, semantic search, D3 visualization
- **DevOps tooling**: ArgoCD, Helm, Kubernetes API, GitHub Actions, Prometheus metrics
